# 犬儒主义
奉行犬儒主义的人通常对其他受野心、欲望、贪婪、物质主义、目标所驱使的人不抱有希望，同时反对将抱负、欲望、贪欲、刺激、功利等作为自己的生活动力（犬儒主义者认为这些都是徒劳，不可实现或毫无意义），因此他们也拒绝财富、权力和荣誉。
